## App setup

Clone the project

```bash
  git clone https://github.com/sergiopm97/portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install project packages

```bash
  npm install
```

Deploy the development version of the project

```bash
  npm run dev
```
